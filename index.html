Welcome to BigFileSharing Pages.

This project started when I needed to transfer a large video file to a friend. There are sites that offer this service but I wanted myself control the deposit of my files. The idea is to do a project with the simplest possible without source code framework and without reinventing the wheel. I use jquery and dropzonejs as third application.

Required

  • a web server (of course!)
  • php 5.6 or up
  • MySQL

Install

Installation is very simple.

  • Transfer the project to the root of your site (i.e: www / bfs)
  • Create a directory that will upload files with write access by www-data (or apache)
  • Setup config.ini
